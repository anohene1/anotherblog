# SomeBlog

## Getting Started

- Install dependencies
```
npm i
```
## Configuring Server

* Clone [server](http://localhost:3000)

* Run server
```
npm run develop
```


* Run App
```bash
npm run dev
```

* Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
